## **Features**

### **Authors API**
- **List all authors** (GET `/api/authors/`)

    - Retrieve the user's favorite books.
    - Get personalized book recommendations based on the favorites.

This application is a Django-based REST API for managing books, authors, and user favorites. It includes features for creating, retrieving, updating, and deleting records of authors and books and also allows users to manage a list of their favorite books and receive recommendations.

---

## **Features**

### **Authors API**
- **List all authors** (GET `/api/authors/`)
- **Create a new author** (POST `/api/authors/`)
- **Retrieve details of an author** (GET `/api/authors/<id>/`)
- **Update an author** (PUT `/api/authors/<id>/`)
- **Delete an author** (DELETE `/api/authors/<id>/`)

### **Books API**
- **List all books** (GET `/api/books/`)
- **Search for books by title** (GET `/api/books/?search=<title>`)
- **Create a new book** (POST `/api/books/`)
- **Retrieve details of a book** (GET `/api/books/<id>/`)
- **Update a book** (PUT `/api/books/<id>/`)
- **Delete a book** (DELETE `/api/books/<id>/`)

### **Favorites API**
- **Add a book to favorites** (POST `/api/favorites/`):
  Add a book to the user's list of favorite books.
    - Required Parameter: `book_id` (Integer)
- **Remove a book from favorites** (DELETE `/api/favorites/`):
  Remove a book from the user's list of favorites.
    - Required Parameter: `book_id` (Integer)
- **Fetch favorites and recommendations** (GET `/api/favorites/`):
    - Retrieve the user's favorite books.
    - Get personalized book recommendations based on the favorites.

---

## **Installation and Setup**

Follow these steps to get the project running locally:

### **1. Clone the Repository**
```bash
git clone <repository_url>
cd <repository_folder>
```

### **2. Create and Activate a Virtual Environment**
```bash
python -m venv env
source env/bin/activate  # On Windows: env\Scripts\activate
```

### **3. Install Dependencies**
Install the required Python packages:
```bash
pip install -r requirements.txt
```

### **4. Apply Migrations**
Set up the database by running the migrations:
```bash
python manage.py makemigrations
python manage.py migrate
```

### **5. Run the Development Server**
Start the server locally:
```bash
python manage.py runserver
```

The application will be available at `http://127.0.0.1:8000/`.

### **Books API**
| Method      | Endpoint                  | Description                     |
|-------------|---------------------------|---------------------------------|
| `GET`       | `/api/books/`             | List all books                 |
| `GET`       | `/api/books/?search=<title>` | Search books by title          |
| `POST`      | `/api/books/`             | Create a new book              |
| `GET`       | `/api/books/<id>/`        | Retrieve details of a book     |
| `PUT`       | `/api/books/<id>/`        | Update a book                  |

### **Favorites API**
| Method      | Endpoint                  | Description                    |
|-------------|---------------------------|--------------------------------|

| `GET`       | `/api/favorites/`         | Fetch favorites and recommendations |

---

## **Data Models**

### **Author**
Represents an author who writes books.
- Fields: `id`, `name`, `bio`

### **Book**
Represents a book written by an author.
- Fields: `id`, `title`, `description`, `author`

### **FavoriteBooks**
Represents a user's list of favorite books.
- Relationships: `user` (OneToOne with `User`), `books` (ManyToMany with `Book`)
